# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**. This project demonstrates fundamental web development concepts such as DOM manipulation, event handling, and game logic implementation.

## 📌 Features

- ✅ Two-player gameplay (X vs O)
- ✅ Automatic winner detection
- ✅ Draw match detection
- ✅ Reset Game functionality
- ✅ New Game option
- ✅ Responsive and interactive UI
- ✅ Dynamic game messages

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**

## 🎯 How to Play

1. Player **X** starts the game.
2. Players take turns marking empty cells.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Click **Reset Game** or **New Game** to play again.
